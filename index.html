Dúvidas frequentes

Esclarecendo suas possíveis dúvidas
Quanto custa o desenvolvimento de cada projeto?
Os projetos são precificados individualmente, com base na quantidade de páginas e no tipo de aplicação desejada. Logo após o briefing, realizamos o orçamento e discutimos melhor os valores. ;)
Desenvolve backend?
Meu foco é desenvolvimento frontend. Apesar disso, elaboro backend de projetos caso o cliente precise de um CRM para o próprio site e não tenha interesse em utilizar wordpress.
Os layouts são feitos do zero?
Sim! Os layouts são elaborados no Figma, utilizando metodologias de UI Design e boas práticas de usabilidade. No entento, podemos discutir caso você já tenha o design do projeto e queira implementar na web.
Como funciona a hospedagem?
Oriento o cliente a comprar previamente o domínio e a hospedagem de sua escolha.
